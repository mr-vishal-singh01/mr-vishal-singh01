# 📚 Learning Resources

## 📖 Recommended Books

### AI & Machine Learning
- **Artificial Intelligence: A Modern Approach** by Russell & Norvig
  - Comprehensive AI fundamentals
  - Covers algorithms and modern techniques

- **Deep Learning** by Goodfellow, Bengio, Courville
  - Neural networks and deep learning
  - Mathematical foundations

- **Python Machine Learning** by Sebastian Raschka
  - Practical ML implementation in Python
  - Hands-on examples

### Cybersecurity & Penetration Testing
- **The Web Application Hacker's Handbook** by Stuttard & Pinto
  - Web security vulnerabilities
  - Testing methodologies

- **Penetration Testing** by Georgia Weidman
  - Complete pentesting guide
  - Real-world scenarios

- **Cryptography Engineering** by Ferguson, Schneier, Kohno
  - Cryptographic systems
  - Security principles

### Systems Programming
- **The C Programming Language** by Kernighan & Ritchie
  - C fundamentals and best practices
  - Essential for systems work

- **Linux Kernel Development** by Robert Love
  - Understanding kernel internals
  - Low-level programming

## 💡 My Learning Strategy

### Principles
1. **Learn by Building** - Apply knowledge to real projects
2. **Read Research** - Stay updated with latest advances
3. **Practice Regularly** - Consistent coding and problem solving
4. **Teach Others** - Solidify understanding through teaching
5. **Stay Curious** - Explore emerging technologies
6. **Combine Domains** - Bridge AI and security knowledge

### Daily Routine
- Read technical news (30 min)
- Code/build projects (2-3 hours)
- Study papers or courses (1 hour)
- Contribute to open source (1 hour)
- Document learnings (30 min)
